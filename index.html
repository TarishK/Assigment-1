Project Idea

Overview

The project idea that I am proposing is a google chrome extension that can be downloaded from the google extension store this extension will work similarly to a scam detector website. The extension will run in the background and deliver the end user with a trust score where 0 is the worst rating and 100 is the best for any website. The extension can be utilised to determine if an individual should continue using the set platform by and can be used by anyone who chooses to browse the web securely. The main purpose behind this idea is to mitigate the risk of a user getting scammed by a malicious website.

Motivation

The main motivation behind this project in fact came from a close family friend who was a victim of an online investment scam. As per the Australian competition & consumer commission (ACCC) Australians have lost more than $1.8 billion in 2021 to scams with true costs being over $2 billion if we consider all the victims that have not officially reported. With Investment scams are the most popular and online shopping scams rating the 7th highest. This chrome extension may not facilitate savings of billions of dollars however having a tool like this readily available is a step in the right directing to provide reassurance to the end user in the case that they are speculating on the legitimacy of a certain platform.

Description

The extension will be user-friendly with a very basic but sleek UI for version 1 the extension will run in the background when the user loads into a new website, like an ad blocker. On the back end, the fundamentals of this extension are based on an API that will be sourced by Apivoid and or a scam adviser. Firstly, the API that is being used will take the domain name and will run checks against domain name blacklist servers. This will help identify potentially malicious and fraudulent websites involved in fake online shopping, malware distribution, and data phishing incidents. In addition, the API will run a WHOIS background check to establish when the set domain was created, any nameservers assigned to the domain, and any domain registrants. All the details complied together will determine the trust score of the website which is then communicated with the software and provided to the end user through the chrome extension.

For convenience, the user does not need to click an additional button or copy and paste the domain name anywhere. The script will be made as a background script to run once the user loads into the set website if the extension is enabled. It will then extract the domain name and feed it into the software which will then provide a trust score. The extension will not prompt the end user if the trust score is equal to or than 80% however, it will provide a small number icon next to the icon badge of the extension to let the user know that their trust score is ready if needed. In the case that the trust score is below 80% the extension will prompt the user with the “trust score + “please review” message. From here on the user can determine if they wish to continue with the website or choose to leave by pressing “leave site” on the UI.

When the user decided to hover above icon badge, they see the UI. The UI itself will be sleek with a light space grey background. At the top it will present the name of the extension “Scoreit” underneath in the middle, trust score of the website will be displayed and at the bottom 2 buttons “Stop extension” “New score”. Both buttons essentially do the exact things they state. On the left the “New score” will run the diagnose of the website again and on the right the “Stop extension” will halt the extension until turned back on by pressing the same button.

Tools and Technologies

The main tools utilised to create this extension will be Atom for the Javascript, HTML & CSS files.
